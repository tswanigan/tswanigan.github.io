  • How To Single Crochet For Beginners!

    • If you don’t already know, there are five basic crochet stitches.
    The single crochet stitch is one of the first stitches you’ll want to learn how to do in crochet. It is the most basic of all basic stitches, yet you can use this single stitch to create so many different things. Here, you will learn via picture tutorial how to single crochet. You will need to know how to make a slip knot and the chain stitch to move forward. Don't fret, found on the website, I linked a number of Youtube videos, I found them to be most helpful! Check there first. When learning to crochet, I prefer to teach my friends with lightly colored yarn and a 5 or 6 mm crochet hook. The light color in yarn helps to see the stitch your hook will go into next.

  • Single Crochet into a Foundation Chain

    • Often, you’ll start a project with chain stitches and a row of single crochet. Here’s how to do that:

    1. Make a slip knot and a foundation chain of 15 stitches.
    2. To make the first single crochet, insert the hook into the second chain from the hook. (Remember, we don’t count the chain that is around the hook.)
    3. Yarn over, and pull up a loop.
    4. Yarn over, and pull through both loops on the hook.
    5. Repeat Steps 2-4 to make a single crochet stitch in each of the remaining 14 chains. Be sure to work in the very last chain – it can be easy to miss.
